## Overview

The Hoodie Boy Timer is a modern, web-based speedcubing timer designed for performance, simplicity, and real training use. It delivers precise timing, a clean interface, and high-quality scramble generation using the Twisty Player API.

Built for cubers who demand focus and efficiency, the project emphasizes a minimal yet powerful experience while maintaining a futuristic and structured design.
